# Echoes of Eolis - User Documentation

This is the user documentation for Echoes of Eolis (version beta-1), a Faxanadu data editor which can be found on its [GitHub repository](https://github.com/kaimitai/faxedit/). It is assumed that users are somwhat acquainted with Faxanadu on the NES.

<hr>

We currently only support editing the US version of the nes-ROM.

The main window for file operations is Project Control.

The data we can edit forms a data hierarchy, from the top-level game metadata down to the individual screens and their sub-data.

* Game Metadata
  * Stages
  * Spawn Points
  * Building Sprite-sets
  * Push-Block
  * Worlds
    * Metatile Definitions
    * Default Palette
    * Mattock Animations
    * Cleanup (utility functions)
    * Screens
      * Screen Tilemaps
      * Screen Sprite-set
      * Screen Doors
      * Screen Scrolling Connections
      * Screen Transitions
